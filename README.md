# mysql_docker

You can run the mysql database in docker easily


**Note** 

I'm using port `3307`

# Quick Start
### Step 1:
Clone the repository
```
git clone <REPO_LINK>
```
```
cd <REPOSITORY>
```

### Step 2:
Start the database
```
docker compose up -d
```

Note:
Default Password: `123`
Default Database: `mydatabase`
